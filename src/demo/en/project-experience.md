2. **MemoryOS Agent Memory System** (AI Agent / Infrastructure)
   Led the design and implementation of a persistent agent memory system, solving context loss in multi-turn conversations and cross-session knowledge retention. Designed a three-layer memory architecture — working memory (sliding window), episodic memory (vector retrieval), and semantic memory (knowledge graph) — with automatic tiered storage and forgetting mechanisms. Innovatively proposed the Memory Router component that automatically routes to the appropriate memory layer based on intent recognition, improving retrieval hit rate from 61% (single-layer) to 92%. The system serves as core infrastructure for the company-wide Agent platform, supporting 120+ agent instances with over 20 million structured memory entries accumulated.
