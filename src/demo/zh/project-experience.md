# Project Experience

1. **AgentForge 多智能体编排平台**（AI Agent/架构）
   从零设计并主导落地企业级多智能体编排平台 AgentForge。提出基于 DAG 的任务编排引擎，支持智能体间依赖声明、并行执行与失败回滚；设计了统一的 Tool Registry 协议，将 200+ 内部 API 封装为标准化工具，实现跨业务线的工具复用率从 12% 提升至 78%。引入 Supervisor-Agent 分层调度架构，通过动态 Prompt 组装与上下文压缩策略，将长链路推理的 Token 消耗降低 43%，端到端延迟降低 35%。平台已接入客服、风控、数据分析等 6 条业务线，日均调度超 50 万次，人工介入率从 32% 降至 7%。
2. **MemoryOS 智能体记忆系统**（AI Agent/基础设施）
   主导设计并实现智能体持久化记忆系统 MemoryOS，解决多轮对话中上下文丢失与跨会话知识沉淀问题。设计了三层记忆架构——工作记忆（滑动窗口）、情景记忆（向量检索）、语义记忆（知识图谱），并实现自动分级存储与遗忘机制。创新性地提出 Memory Router 组件，基于意图识别自动路由至对应记忆层，检索命中率从单层方案的 61% 提升至 92%。系统已作为核心基础设施接入全公司 Agent 平台，支撑 120+ 智能体实例，累计沉淀结构化记忆条目超 2000 万条。
3. **CodePilot 研发智能体**（AI Agent/应用）
   独立架构并落地面向软件研发全流程的智能体 CodePilot，覆盖需求拆解、代码生成、CR 自动化、故障诊断四大场景。设计了 ReAct + Plan-and-Execute 混合推理框架，针对复杂需求自动拆解为可执行子任务链，并引入 Self-Reflection 机制实现生成代码的迭代自纠。将代码仓库索引、CI/CD 流水线、监控告警等 15 项研发工具封装为 MCP Tool，实现智能体对研发环境的深度感知与操作闭环。上线后代码生成采纳率达 41%，CR 自动审查覆盖率达 68%，故障定位平均耗时从 45 分钟缩短至 8 分钟。
